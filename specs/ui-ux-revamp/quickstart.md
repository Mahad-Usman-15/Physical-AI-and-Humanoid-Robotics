# Quickstart

This document provides instructions on how to run the Docusaurus development server to preview the styling changes.

## Prerequisites

-   Node.js (version >= 20.0)
-   npm

## Running the Development Server

1.  Navigate to the `physicalroboticbook` directory:
    ```bash
    cd physicalroboticbook
    ```

2.  Install the dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm start
    ```

The website will be available at `http://localhost:3000`. Changes to the source code will be reflected live in the browser.
